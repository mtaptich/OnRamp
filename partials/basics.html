Bostock curates a collection of d3 examples, including several very basic ones at d3js.org. Visit there, click on Examples, and scroll down or search for "Basic Charts."

We are going to copy-and-paste the code for a line chart. Make a folder on your computer, and use Sublime Text to save a blank document there called index.html. Then save another blank document called data.tsv in the same folder.

Next, copy the code section from the Line Chart example Page and paste it into the index.html file and save it. Finally, copy the data and paste it into data.tsv.

Now run your localhost like you dide before using http-server and navigate to the folder you made. You should see the line chart.

If you see a beautiful visualization on any other site, then you can right-click and hit View Page Source or View Frame Source to see the code that generated it. Here is one of my visualizations. Use View Frame Source on this guy, because it is embedded in this website like a little window to another website.
